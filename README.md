# E-commerce-Sales-Performance-Analysis
This project presents an in-depth analysis of e-commerce sales data using Microsoft Excel. The goal is to uncover meaningful insights into business performance and support data-driven decision-making through an interactive dashboard.
The dataset consists of 5,000 records and 14 features, including sales, profit, product categories, regions, and payment methods. Using Excel’s data analysis and visualization tools, the data is transformed into a dynamic and easy-to-understand dashboard.
Key Features
Interactive dashboard built entirely in Excel
KPI tracking: Total Sales, Profit, Quantity, and Orders
Sales analysis by region using donut charts
Category and sub-category performance using bar charts
Payment mode distribution using pie charts
Monthly sales trends visualized with line charts
Use of Pivot Tables, Pivot Charts, and slicers for interactivity
Key Insights
High sales performance with relatively lower profit margins
Balanced sales distribution across regions
Furniture and home-related products dominate sales
Equal preference for different payment methods
Stable and consistent monthly sales trends
Tools & Techniques
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning & Transformation
Dashboard Design & Data Visualization
Slicers and Filters for interactivity
Conclusion
The dashboard provides a clear overview of business performance and highlights areas for improvement. While sales are strong, focusing on profit optimization and strategic marketing can further enhance business growth.
